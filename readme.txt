Remind Buddypress user with all required field that need to be filled.

== Description ==

This free plugin will notify and redirect to profile edit page with all its required fields

== Installation ==

* Upload `bp-required-field-reminder` to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
